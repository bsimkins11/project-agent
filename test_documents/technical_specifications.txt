Project Agent Technical Specifications

SYSTEM REQUIREMENTS

Minimum System Requirements:
- CPU: 2 cores, 2.4 GHz
- RAM: 4 GB
- Storage: 20 GB available space
- Network: Broadband internet connection

Recommended System Requirements:
- CPU: 4 cores, 3.0 GHz or higher
- RAM: 8 GB or more
- Storage: 50 GB SSD
- Network: High-speed broadband (100 Mbps+)

SUPPORTED FILE FORMATS

Document Types:
- PDF: Adobe Portable Document Format
- DOCX: Microsoft Word documents
- TXT: Plain text files
- MD: Markdown documents
- HTML: HyperText Markup Language

File Size Limits:
- Maximum file size: 10 MB per document
- Batch upload: Up to 50 files per batch
- Total storage: Unlimited (subject to GCP quotas)

IMAGE PROCESSING

Supported Image Formats:
- JPEG, PNG, GIF, BMP, TIFF
- Maximum resolution: 4096x4096 pixels
- OCR processing for text extraction
- Automatic thumbnail generation

API ENDPOINTS

Core API Endpoints:
- POST /chat - Process chat queries
- GET /inventory - List available documents
- GET /documents/{id} - Get document details
- POST /upload - Upload new documents
- POST /admin/ingest/csv - Bulk CSV ingestion
- POST /admin/ingest/link - Process web links

Response Formats:
- JSON for all API responses
- Standardized error codes
- Pagination support for large datasets
- CORS enabled for web applications

DATABASE SCHEMA

Document Metadata:
- id: Unique document identifier
- title: Document title
- type: Document type (PDF, DOCX, etc.)
- size: File size in bytes
- uri: GCS storage URI
- status: Processing status
- upload_date: ISO timestamp
- processing_result: AI processing data

Vector Search Index:
- vector_id: Unique vector identifier
- embedding: 768-dimensional vector
- metadata: Document and chunk information
- doc_id: Reference to source document
- chunk_text: Text content for this chunk
